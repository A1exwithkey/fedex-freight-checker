# V2.1.0-excel-ui-reference

日期：2026-05-16

## 本版目标

在不改核心计算逻辑的前提下，把 Excel 调整成更像内部工具的第一版：打开就是计算器，中文优先展示，数据表仍保留适合后端读取的结构。

## 主要变化

- `calculator` 调整为第一个 sheet。
- 计算器中文化：输入项、输出项、中文说明、字段名、公式/规则分列。
- 高亮最终 USD 结果。
- 调整字体、列宽、边框、冻结窗格和表头样式。
- `country_zone_ip` Excel 展示新增中文优先显示列。
- 新增 `docs/online_research_notes.md`，记录外部工具经验。

## 未改变

- IP 包裹固定费率数据未改变。
- 21kg+ 每公斤费率数据未改变。
- 国家/地区 IP Zone 数据未改变。
- Final USD 公式未改变。
