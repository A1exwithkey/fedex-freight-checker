# V2.3.0-demand-surcharge

日期：2026-05-16

## 本版目标

在 FedEx IP 运费核价助手中加入旺季附加费，也就是 FedEx Demand Surcharge。

## 计算逻辑

```text
Demand Surcharge CNY = MAX(weight_kg × demand_rate_cny_per_kg, demand_minimum_cny)
Freight Before Fuel CNY = Base Freight CNY + Demand Surcharge CNY
Final USD = Freight Before Fuel CNY × (1 + Fuel Rate) × Redundancy Factor ÷ Exchange Rate
```

默认：

- Fuel Rate = 48%
- Redundancy Factor = 1.1
- Exchange Rate = 6.8
- Demand Minimum = RMB 7.2/票

## 数据来源

FedEx Demand Surcharge from October 28, 2024 (Mainland China)  
https://www.fedex.com/content/dam/fedex/international/rates/fedex-ds-en-cn-2024oct28.pdf
