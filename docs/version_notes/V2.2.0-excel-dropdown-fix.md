# V2.2.0-excel-dropdown-fix

日期：2026-05-16

## 本版目标

解决 Excel 打开兼容性和计算器可用性问题：减少可能触发修复的复杂公式，增加中文下拉选择，并把默认燃油附加费率改为 48%。

## 主要变化

- 计算器保留在第一个 sheet。
- 新增中文目的地下拉。
- 新增手动输入目的地；手动输入优先于下拉。
- 默认燃油附加费率改为 48%。
- “安全系数”改名为“冗余系数”。
- 字体改为 Calibri 风格。
- 公式改为 `VLOOKUP` / `SUMIFS`，避免 `XLOOKUP` / `LET` 的兼容风险。
- 新增隐藏 sheet `country_dropdown` 作为下拉数据源。
- 中文国家名去掉 PDF 换行造成的内部空格。

## 输出文件

`outputs/运费核价助手_FedEx_IP_V2.2_下拉修正版.xlsx`

## 注意

如果 Excel 仍提示修复，优先检查数据验证命名区域和 Excel 版本；当前文件已通过 zip 结构校验。
