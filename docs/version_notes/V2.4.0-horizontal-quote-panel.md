# V2.4.0-horizontal-quote-panel

日期：2026-05-16

## 本版目标

把计算器改成销售更容易看的横向报价面板：上面输入，下面一排展示 IP 分区、旺季大区、费率和价格。

## 主要变化

- 目的地下拉按英文 A-Z 排序。
- 下拉显示格式改为 `English (中文)`。
- 剔除美国西部州/邮编行，保留美国西部汇总项。
- 顶部横向输入：目的地、手输、重量、燃油、冗余、汇率、旺季最低收费。
- 横向输出：匹配国家、IP 分区、旺季大区、旺季费率、基础运费、旺季附加费、最终 USD。
- 字段名和公式行隐藏。
- 旺季最低收费默认空白；空白时不套最低收费。
- 美国西部旺季区域修正为北美区。

## 输出文件

`outputs/运费核价助手_FedEx_IP_V2.4_横向报价版.xlsx`
