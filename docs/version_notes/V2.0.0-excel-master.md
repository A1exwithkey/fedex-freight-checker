# V2.0.0-excel-master

日期：2026-05-16

## 本版目标

把 FedEx 运费核价助手从单个 Excel 文件整理成可维护的小型内部工具项目，并生成可复跑的 Excel 母版。

## 数据来源

- 原始 PDF：`data_raw/FEDEX报价表-2026.1.5生效.pdf`
- 原始 V1 Excel：`data_raw/运费核价助手_FedEx_IP_V1_提取版.xlsx`

## 输出

- `data_processed/fedex_ip_data.json`
- `data_processed/country_zone_ip.csv`
- `data_processed/ip_parcel_rate_0_20_5kg.csv`
- `data_processed/ip_parcel_rate_21kg_plus.csv`
- `data_processed/country_alias.csv`
- `data_processed/validation_checks.csv`
- `outputs/运费核价助手_FedEx_IP_V2_优化版.xlsx`

## 校验结论

- IP 包裹固定费率：943 行，完整。
- IP 21kg+ 每公斤费率：161 行，完整。
- V1 Excel 费率表与 PDF 抽取值对比：0 个差异。
- V1 Excel 国家分区表有 14 行换行粘连风险，V2 已重新解析。

## 已知风险

- 美国西部 Zone 1 目前需要明确输入美国西部；尚未按邮编自动判断。
- 国家别名覆盖常见输入，不保证覆盖所有销售习惯写法。
- 当前 Excel 使用现代函数 `XLOOKUP`、`LET`，旧版 Excel 可能不支持。
