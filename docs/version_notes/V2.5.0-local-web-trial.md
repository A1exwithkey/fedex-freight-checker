# V2.5.0-local-web-trial

日期：2026-05-17

## 本版目标

把项目推进到“Excel 计算母版 + Streamlit 本地网页试算版”。Excel 继续作为可复核数据母版，网页用于内部试算体验。

## 数据变化

- 需求附加费来源改为 `FEDEX需求附加费-2026.4.13日生效至另行通知.pdf`。
- 只读取“中国大陆出口的国际货件”列。
- 需求附加费区域费率：
  - 澳大利亚、新西兰：0
  - 亚洲：0
  - 美国和波多黎各：0
  - 加拿大：0
  - 以色列：24.0
  - 欧洲：8.0
  - 印度：0
  - MEISA：11.2
  - 墨西哥：0
  - LAC：0
- 最低收费规则：费率大于 0 时按 `MAX(weight × rate, 1.80)`，费率为 0 时需求附加费为 0。

## 界面变化

- Excel `calculator` 改为输入区、自动匹配区、报价结果、版本信息四块。
- Streamlit 网页支持目的地下拉、手输、重量、燃油附加费、冗余系数和汇率。
- 网页和 Excel 都显示：
  - 工具版本：2026-05-17
  - IP 协议价：2026-01-05
  - 需求附加费：2026-04-13
  - 燃油费：人工维护，默认 48%

## 风险点

- Netherlands Antilles、Syrian Arab Republic、Yemen 未在 2026-04-13 需求附加费 PDF 区域脚注中精确匹配，继续保留 `Need Review`。
- 2026-05-17 检索 FedEx 公开页面时可见 2026-05-11 生效的需求附加费版本；本版仍按用户提供的 2026-04-13 PDF 执行，后续需业务确认是否切换。
