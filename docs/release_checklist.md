# Release Checklist

每次准备给销售试用或更新网页前，按这份清单检查。

## 数据源

- [ ] IP 协议价 PDF 文件名和生效日期正确。
- [ ] 需求附加费 PDF 文件名和生效日期正确。
- [ ] 需求附加费只使用“中国大陆出口的国际货件”列。
- [ ] 燃油费显示为当前确认值，或明确写“暂定”。
- [ ] 汇率和冗余系数默认值符合当前业务口径。

## 业务范围

- [ ] 只做中国 FedEx 国际出口。
- [ ] 只做 IP。
- [ ] 只做包裹。
- [ ] 不包含 IPE、IE、进口、第三方支付、重货、快递封、快递袋。
- [ ] 不包含税费、偏远地区附加费、特殊处理费。

## 关键规则

- [ ] 美国默认 `USA / United States / 美国` 映射到美国其他地区 Zone 2。
- [ ] 美国西部必须人工选择 `United States - Western Region (美国西部)` 才映射 Zone 1。
- [ ] 0.5kg-20.5kg 使用固定费率。
- [ ] 21kg 及以上使用实际重量乘以每公斤费率。
- [ ] 需求附加费费率为 0 时，需求附加费为 0。
- [ ] 需求附加费费率大于 0 时，按最低收费规则计算。

## 验证样例

- [ ] 美国其他地区 10kg。
- [ ] 美国西部 10kg。
- [ ] 德国 10kg。
- [ ] 肯尼亚 10kg。
- [ ] 以色列 10kg。
- [ ] 澳大利亚 20.5kg。
- [ ] 德国 21kg。
- [ ] 新加坡 500kg。

## 网页检查

- [ ] `streamlit run app/streamlit_app.py` 可以启动。
- [ ] 页面顶部版本日期正确。
- [ ] 目的地下拉可以选择。
- [ ] 手输目的地可以匹配到对应下拉。
- [ ] 报价结果显示最终 USD、基础运费、需求附加费、燃油附加费、含冗余金额。
- [ ] 自动匹配区字段简洁，没有重复字段。
- [ ] Need Review 时不会继续假装报价成功。

## Excel 检查

- [ ] `scripts/02_build_excel.py` 可以生成 Excel。
- [ ] Excel 打开不弹修复。
- [ ] `calculator` 位于第一个 sheet。
- [ ] 公式可复核。
- [ ] `validation_checks` 已更新。

## 发布记录

- [ ] `CHANGELOG.md` 已更新。
- [ ] `docs/version_notes/` 已新增本版说明。
- [ ] Git commit 已创建。
- [ ] 如已部署网页，记录访问地址和部署方式。
